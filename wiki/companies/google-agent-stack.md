## 这是什么

这里说的 `Google agent stack`，不是单一产品，而是 Google / DeepMind 最近公开信号里逐渐浮现的 `Gemini + Gemma + realtime multimodal + science + safety` 组合。

## 主要组成

- `Gemini 3 Deep Think`
- `Gemini 3.1 Flash-Lite`
- `Gemini 3.1 Flash Live`
- `Gemma 4`
- `Project Genie`
- `Lyria 3`
- `harmful manipulation eval toolkit`

## 为什么重要

Google 的路线和 OpenAI、Anthropic 不一样：

- 没有把所有叙事都压到“开发者 agent runtime”
- 而是在同时推进：
  - `frontier reasoning`
  - `science / research`
  - `open models`
  - `real-time multimodal`
  - `consumer product surfaces`

## 当前判断

### 1. Google 的 agent 主线更分散，但技术面更宽

- `Gemini` 负责闭源旗舰和实时多模态
- `Gemma` 负责开放模型
- `DeepMind` 负责 research roadmap 和 safety toolkit

### 2. 它的竞争点不只是开发者工作流

还包括：

- world model
- science
- music / audio
- open model distribution

### 3. 这条线值得长期保留

如果只看“谁的 coding agent 更强”，会低估 Google 的长期方向。

## 还要继续观察什么

- `Gemma` 是否会继续向 agent workflow 深化
- `Flash Live` 是否会把语音交互真正推进成默认入口
- `science / world model` 是否会反向进入更通用的开发者平台
